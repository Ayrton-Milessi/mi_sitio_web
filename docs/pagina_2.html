🌐 Análisis de registros de servidor web

Período: Septiembre 2025 - Octubre 2025

Herramientas: Grafana, Python, PostgreSQL, ETL, Data Warehouse

En conjunto con mis compañeros, diseñamos e implementamos una arquitectura para analizar datos de archivos logs de servidores Apache.

  • Implementamos un Data Warehouse (esquema estrella) para almacenar y consultar los datos de manera eficiente.
  • Desarrollamos un proceso ETL para asegurar la calidad de los datos haciendo la limpieza y transformación necesarias.
  • También detectamos los bots dentro de los registros para mejorar la precisión del análisis.
  • Aplicamos un algoritmo de Minería de Datos para encontrar patrones en la navegación de los usuarios.
  • Conectamos nuestro DW con Grafana para crear dashboards interactivos y visualizaciones de alto valor.
